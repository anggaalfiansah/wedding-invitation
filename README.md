## Run Locally

Clone the project at https://github.com/anggaalfiansah/wedding-invitation.git

Go to the project directory

Install dependencies

```bash
  yarn install
```

run program

```bash
  yarn start
```

build

```bash
  yarn run build
```

preview build

```bash
  yarn run preview
```
